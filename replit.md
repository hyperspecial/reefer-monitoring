# Reefer Monitoring System

An educational, polyglot implementation of an IoT-enabled Reefer (refrigerated shipping container) Monitoring System. Demonstrates telemetry tracking, historical logging, and alert systems across four programming languages.

## Project Structure

| File | Language | Focus |
|------|----------|-------|
| `reefer-monitoring.js` | Node.js | Event-Driven Architecture |
| `reefer_monitoring.py` | Python | System Design & Data Modeling |
| `ReeferMonitoring.java` | Java | Object-Oriented Programming |
| `reefer_monitoring.cpp` | C++ | Memory Efficiency & Performance |

## Running Each Implementation

### Node.js (default workflow)
```bash
node reefer-monitoring.js
```

### Python
```bash
python3 reefer_monitoring.py
```

### Java
```bash
javac ReeferMonitoring.java
java ReeferMonitoring
```

### C++
```bash
g++ reefer_monitoring.cpp -o reefer_app
./reefer_app
```

## User Preferences

- No frontend; all implementations are console-based scripts.
